# GA 프로모션 ON AX 고객 정의

## 1. 목적

이 문서는 `GA 프로모션 ON AX` 대시보드를 사용하는 주요 사용자와 실제 업무 목적을 정리한 기준 문서다.  
현재 기준 기능은 `로그인 → 프로모션 현황 조회 → 관리자 업로드/안내장 생성/전체 승인 → 다운로드` 흐름이다.

## 2. 핵심 사용자

### 2-1. 본사담당자

- 역할: 관리자
- 주요 업무:
  - 월별 프로모션 엑셀 업로드
  - 권한 사번 엑셀 업로드
  - 기타자료실 파일 업로드
  - 표준안내장 생성
  - 전체 승인
- 기대 결과:
  - 업로드한 데이터가 즉시 대시보드에 반영된다.
  - 확정건만 안내장 생성 및 다운로드 가능 상태가 된다.
  - 월별 이력이 유지된다.

### 2-2. 영업관리자

- 역할: 조회 사용자
- 주요 업무:
  - 지역단/지점/GA지사 기준으로 현황 조회
  - 확정/미확정 상태 확인
  - 확정건 안내장 미리보기 및 다운로드
  - 전체대리점 현황, 상품별 예상해약환급금 다운로드
- 기대 결과:
  - 자기 조직 기준으로 빠르게 찾을 수 있다.
  - 관리 메뉴 없이 조회와 다운로드에 집중할 수 있다.

## 3. 사용자 문제

### 본사담당자 문제

- 월별 프로모션 데이터를 수작업으로 정리하고 전달하는 시간이 오래 걸린다.
- 지사별 안내장 형식과 문구가 매번 달라져 관리가 어렵다.
- 확정/미확정 기준이 흩어져 있어 승인 관리가 번거롭다.
- 조직 정보와 프로모션 데이터가 따로 있어 조회 화면 불일치가 발생할 수 있다.

### 영업관리자 문제

- 자신이 봐야 하는 지점/GA지사 현황을 한 번에 찾기 어렵다.
- 확정된 건만 내려받아야 하는데 상태 판단이 어렵다.
- 자료실 파일과 프로모션 현황이 분리되어 있으면 업무 흐름이 끊긴다.

## 4. 사용 시나리오

### 시나리오 A. 본사담당자 월별 반영

1. 본사담당자가 5자리 사번으로 로그인한다.
2. 관리자페이지에서 월별 프로모션 엑셀을 업로드한다.
3. 조직 시트 기준으로 지역단/지점/GA지사 정보가 매칭된다.
4. 표준안내장을 생성한다.
5. 전체 승인한다.
6. 확정건은 다운로드 가능, 미확정건은 비활성 상태가 된다.

### 시나리오 B. 영업관리자 조회

1. 영업관리자가 5자리 사번으로 로그인한다.
2. 지역단/지점/GA지사로 검색한다.
3. 상태와 안내장 가능 여부를 확인한다.
4. 확정건만 미리보기 또는 다운로드한다.

### 시나리오 C. 자료실 운영

1. 본사담당자가 전체대리점 현황 파일을 업로드한다.
2. 본사담당자가 상품별 예상해약환급금 파일을 업로드한다.
3. 사용자 화면 상단/자료실 섹션에서 즉시 다운로드 가능 상태가 된다.

## 5. 반드시 만족해야 하는 기준

- 로그인은 `사번 5자리 = 비밀번호 5자리` 규칙을 따른다.
- 본사담당자만 관리자페이지를 본다.
- 영업관리자는 관리자 기능을 보지 않는다.
- 미확정건은 안내장 다운로드가 비활성화된다.
- 지역단은 조직 시트와 데이터 시트의 관리지점 매칭 기준으로 노출된다.
- 월별 업로드 데이터는 덮어쓰기만 하지 말고 월 기준으로 유지된다.

## 6. 성공 기준

- 본사담당자가 별도 정리 없이 엑셀 업로드 후 현황 반영까지 한 화면에서 처리할 수 있다.
- 영업관리자가 1분 안에 자기 지점의 상태와 다운로드 가능 여부를 확인할 수 있다.
- 확정/미확정/오류 기준이 화면에서 일관되게 보인다.
