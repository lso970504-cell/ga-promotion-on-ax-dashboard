# GA 프로모션 ON AX PRD

## 1. 제품 개요

`GA 프로모션 ON AX`는 본사담당자와 영업관리자가 월별 GA 프로모션 현황을 조회하고, 업로드한 엑셀을 기준으로 안내장을 생성·승인·다운로드할 수 있도록 만든 운영형 대시보드다.

## 2. 배경

현재 업무는 다음 문제가 있다.

- 프로모션 현황이 엑셀 중심이라 조회가 느리다.
- 조직 기준 지역단/지점/GA지사 매칭이 수작업에 의존한다.
- 안내장 생성 여부와 승인 상태가 분리되어 있다.
- 확정/미확정 관리가 일관되지 않다.
- 기타자료실 파일도 별도로 관리되어 사용자 동선이 분산된다.

## 3. 목표

### 목표 1

본사담당자가 엑셀 업로드부터 안내장 생성, 전체 승인까지 한 화면에서 처리할 수 있게 한다.

### 목표 2

영업관리자가 지역단/지점/GA지사 기준으로 빠르게 조회하고 확정건만 다운로드할 수 있게 한다.

### 목표 3

월별 업로드 데이터를 유지하여 이전 월과 현재 월을 구분해서 조회할 수 있게 한다.

## 4. 비목표

- 자동 최종 승인
- 외부 사용자 공개 서비스
- 모바일 앱 별도 개발
- 실시간 다중 사용자 동시 편집
- PPT 자동 생성의 정식 운영화

## 5. 사용자

### 본사담당자

- 관리자 기능 사용
- 업로드/생성/승인 담당

### 영업관리자

- 조회 및 다운로드 담당
- 관리자 기능 비노출

## 6. 핵심 기능

### 6-1. 로그인

- 사번 5자리 입력
- 비밀번호는 동일한 5자리
- 권한 사번 업로드 목록 반영
- 역할 구분: `hq`, `branch`

### 6-2. 프로모션 현황 조회

- 월 선택
- 지역단 필터
- 지점 필터
- GA지사 검색
- 상태 정렬
- 확정/미확정/오류 표시

### 6-3. 관리자페이지

- 프로모션 업로드
- 표준안내장 생성
- 전체 승인
- 권한 사번 업로드
- 기타자료실 업로드

### 6-4. 안내장 처리

- 확정건만 템플릿 생성 대상
- 전체 승인 후 다운로드 가능
- 미확정건은 다운로드 비활성화
- 미리보기 제공

### 6-5. 기타자료실

- 전체대리점 현황 파일 업로드/다운로드
- 상품별 예상해약환급금 엑셀 업로드/다운로드

## 7. 상태 규칙

### 확정

- 필수 데이터 존재
- 템플릿 생성 가능

### 미확정

- 프로모션 수치 데이터 미입력
- 다운로드 비활성화

### 오류

- 조직 매핑 실패
- 업로드 구조 확인 필요

## 8. AS-IS / TO-BE

### AS-IS

- 엑셀 수작업 의존
- 승인/배포 흐름 분리
- 자료실과 현황이 분리 운영
- 지역단/지점 정합성 확인이 번거로움

### TO-BE

- 엑셀 업로드 즉시 화면 반영
- 조직 시트 기반 자동 매핑
- 안내장 생성/전체 승인/다운로드 연결
- 자료실도 같은 화면에서 운영

## 9. 화면 요구사항

### 로그인 페이지

- 심플한 1페이지 로그인
- 큰 제목
- 5자리 사번 규칙 안내

### 프로모션 현황 페이지

- 큰 제목과 월 기준 탭
- 검색 필터 1행 정렬
- 테이블 스크롤 가능
- 상태 중앙 정렬
- 안내장 버튼은 `미리보기`, `다운로드`

### 관리자페이지

- KPI 카드
- 업로드/생성/승인/권한/자료실 업로드 구성
- 카드 간 여백과 정렬 일관성 유지

## 10. 데이터 요구사항

### 필수 입력

- 프로모션 업로드 엑셀
- 조직 시트
- 권한 사번 엑셀

### 권장 입력

- 유의사항 시트
- 기타자료실 업로드 파일

## 11. 완료 기준

- 본사담당자가 엑셀 업로드 후 프로모션 현황이 갱신된다.
- 지역단/지점/GA지사 값이 조직 기준으로 맞게 노출된다.
- 표준안내장 생성 버튼이 확정건 기준으로 동작한다.
- 전체 승인 후 확정건만 다운로드 가능하다.
- 영업관리자 로그인 시 관리자페이지가 숨겨진다.

## 12. 향후 확장

- Supabase 저장 전환
- 월별 업로드 이력 서버 저장
- 승인 로그 관리
- 템플릿 버전 관리
- 이미지/PPT 자동 생성 고도화
