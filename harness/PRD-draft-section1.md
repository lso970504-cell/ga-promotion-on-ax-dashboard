## PRD 초안 - Section 1

### 제품명

GA 프로모션 ON AX 대시보드

### Beachhead 사용자

- 본사 프로모션 운영 담당자
- 영업관리자 및 지사 조회 사용자

### 핵심 사용자 문제

- 계속 갱신되는 엑셀 데이터를 기준으로 어떤 건이 확정인지 통합 관리하기 어렵다.
- 지사별 표준안내장을 반복 수작업으로 만들면 시간이 많이 들고 오류가 생기기 쉽다.
- 현업 사용자는 다운로드 가능 여부를 빠르게 알고 싶어 한다.

### 1차 핵심 시나리오

1. 관리자가 엑셀 파일을 업로드한다.
2. 시스템이 맞는 행만 반영하고 오류 행은 별도 표시한다.
3. 정상건만 표준안내장 초안을 만든다.
4. 관리자가 행별 미리보기로 초안을 확인한다.
5. 관리자가 `전체 승인`을 누른다.
6. 사용자 화면에서는 `확정` 건만 이미지 다운로드가 가능하다.

### 1차 성공 모습

- 운영자가 수작업 안내장 제작을 줄인다.
- 사용자 화면에서 `확정 / 미확정 / 오류` 상태가 명확하게 보인다.
- 확정 건만 표준양식 이미지로 다운로드된다.

### 보류 항목

- 문구 수정 기능
- PPT 생성
- DB 저장
- 세부 승인 이력 관리
