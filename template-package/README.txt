업로드/전달 방법

1. template-package 폴더 안의 파일 3개를 항상 같이 보관합니다.
   - promo-template.html
   - promo-template-data.browser.js
   - xlsx.full.min.js

추가 샘플 파일
- promo-template-upload-sample.xlsx
  이 파일을 템플릿 화면의 엑셀 업로드 버튼으로 올리면
  어떤 형식으로 입력해야 하는지 바로 확인할 수 있습니다.

2. 다른 PC나 웹하드에 올릴 때는 파일 1개만 올리지 말고
   template-package 폴더 전체를 그대로 복사하거나 업로드하면 됩니다.

3. 실행 방법
   - promo-template.html 파일을 엽니다.
   - 화면 오른쪽 편집 패널에서 제목, 지사명, 숫자를 수정할 수 있습니다.
   - 엑셀 업로드 버튼으로 데이터 파일을 올리면 표 값이 자동 반영됩니다.

4. 인쇄 방법
   - Ctrl+P 를 누르면 편집 패널은 자동으로 숨겨집니다.
